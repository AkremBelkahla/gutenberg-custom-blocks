Custom Gutenberg blocks for WordPress editor.

== Description ==

Ce plugin ajoute des blocs Gutenberg personnalisés à l'éditeur WordPress, améliorant votre expérience de création de contenu.

Fonctionnalités :
* Bloc de texte et d'image personnalisé
* Bloc d'appel à l'action
* Bloc de témoignage

== Installation ==

1. Téléchargez les fichiers du plugin dans le répertoire `/wp-content/plugins/gutenberg-custom-blocks`, ou installez le plugin directement depuis l'écran des plugins de WordPress.
2. Activez le plugin via l'écran 'Extensions' de WordPress.
3. Utilisez les nouveaux blocs dans l'éditeur Gutenberg.

== Foire Aux Questions ==

= Ces blocs sont-ils compatibles avec WordPress 6.0 et versions ultérieures ? =

Oui, tous les blocs sont conçus pour fonctionner avec WordPress 6.0 et versions ultérieures.
